## [Box Shadow](https://tailwindcss.com/docs/box-shadow)
The `boxShadow` property allows you to control the shadow effect applied to elements. Each shadow is specified with various parameters that define its appearance.
<Source language='js' code={`boxShadow: ${JSON.stringify(component.boxShadow, null, 2)}`} />

### `sm`
A minimal shadow effect, perfect for subtle elevation.
<Source language="tsx" code={`sm: ${component.boxShadow.sm}`} />
- `0`: Horizontal offset (no horizontal shift).
- `1px`: Vertical offset (shadow moves `1px` down from the element).
- `2px`: Blur radius (the shadow becomes more blurred with this value).
- `0`: Spread radius (no change to the shadowâ€™s size).
- `rgb(0 0 0 / 0.05)`: Color with `5%` opacity.

### `DEFAULT`
Combines two shadows for a balanced medium effect.
<Source language="tsx" code={`DEFAULT: ${component.boxShadow.DEFAULT}`} />
`0 1px 3px 0 rgb(0 0 0 / 0.1)`: 
- `0`: Horizontal offset.
- `1px`: Vertical offset.
- `3px`: Blur radius.
- `0`: Spread radius.
- `rgb(0 0 0 / 0.1)`: Color with `10%` opacity.

`0 1px 2px -1px rgb(0 0 0 / 0.1)`:
- `0`: Horizontal offset.
- `1px`: Vertical offset.
- `2px`: Blur radius.
- `-1px`: Spread radius (reduces the size of the shadow).
- `rgb(0 0 0 / 0.1)`: Color with `10%` opacity.

### `md`
Provides a medium shadow effect with added depth.
<Source language="tsx" code={`md: ${component.boxShadow.md}`} />

`0 4px 6px -1px rgb(0 0 0 / 0.1)`:
- `0`: Horizontal offset.
- `4px`: Vertical offset.
- `6px`: Blur radius.
- `-1px`: Spread radius.
- `rgb(0 0 0 / 0.1)`: Color with `10%` opacity.

`0 2px 4px -2px rgb(0 0 0 / 0.1)`:
- `0`: Horizontal offset.
- `2px`: Vertical offset.
- `4px`: Blur radius.
- `-2px`: Spread radius.
- `rgb(0 0 0 / 0.1)`: Color with `10%` opacity.

### `lg`
Creates a more pronounced shadow effect with greater depth.
<Source language="tsx" code={`lg: ${component.boxShadow.lg}`} />

`0 10px 15px -3px rgb(0 0 0 / 0.1)`:
- `0`: Horizontal offset.
- `10px`: Vertical offset.
- `15px`: Blur radius.
- `-3px`: Spread radius.
- `rgb(0 0 0 / 0.1)`: Color with `10%` opacity.

`0 4px 6px -4px rgb(0 0 0 / 0.1)`:
- `0`: Horizontal offset.
- `4px`: Vertical offset.
- `6px`: Blur radius.
- `-4px`: Spread radius.
- `rgb(0 0 0 / 0.1)`: Color with `10%` opacity.

### `xl`
Offers a strong shadow effect for high elevation.
<Source language="tsx" code={`xl: ${component.boxShadow.xl}`} />

`0 20px 25px -5px rgb(0 0 0 / 0.1)`:
- `0`: Horizontal offset.
- `20px`: Vertical offset.
- `25px`: Blur radius.
- `-5px`: Spread radius.
- `rgb(0 0 0 / 0.1)`: Color with `10%` opacity.

`0 8px 10px -6px rgb(0 0 0 / 0.1)`:
- `0`: Horizontal offset.
- `8px`: Vertical offset.
- `10px`: Blur radius.
- `-6px`: Spread radius.
- `rgb(0 0 0 / 0.1)`: Color with `10%` opacity.

### `2xl`
A very large shadow with substantial depth, suitable for prominent elements.
<Source language="tsx" code={`'2xl': ${component.boxShadow['2xl']}`} />

- `0`: Horizontal offset.
- `25px`: Vertical offset.
- `50px`: Blur radius.
- `-12px`: Spread radius.
- `rgb(0 0 0 / 0.25)`: Color with `25%` opacity.

### `inner`
Creates a subtle inner shadow effect, giving the appearance of the content being pressed into the element.
<Source language="tsx" code={`inner: ${component.boxShadow.inner}`} />
- `inset`: Indicates that the shadow is applied inside the element.
- `0`: Horizontal offset.
- `2px`: Vertical offset.
- `4px`: Blur radius.
- `0`: Spread radius.
- `rgb(0 0 0 / 0.05)`: Color with `5%` opacity.

### `none`
Removes any shadow effect, resulting in a flat appearance.
<Source language="tsx" code={`none: ${component.boxShadow.none}`} />
