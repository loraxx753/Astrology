# [Filters](https://tailwindcss.com/docs/filter) <small className='text-xs'>[Jump to source](#source-code)</small>
The `filters` property allows you to control various filter effects applied to elements. Each filter is specified with various parameters that define its appearance.

## `backdropBlur`
Applies a blur effect to the backdrop of an element.
<Source language="tsx" code={`backdropBlur: ${component.backdropBlur}`} />

## `dropShadow`
Defines the drop shadow effects for various levels of intensity.
<Source language="tsx" code={`dropShadow: ${JSON.stringify(component.dropShadow, null, 2)}`} />
- `sm`: Minimal shadow effect.
- `DEFAULT`: Balanced shadow effect.
- `md`: Medium shadow effect.
- `lg`: Pronounced shadow effect.
- `xl`: Strong shadow effect.
- `2xl`: Very large shadow effect.
- `none`: No shadow effect.
